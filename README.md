# GitHub-Blogger

A blogging website powered by GitHub, built with Electron and styled similar to github.blog.

## Features

- 🎨 GitHub Blog-inspired design
- 📱 Responsive layout for iPhone and iPad
- ⚡ Built with Electron for cross-platform desktop support
- 🎯 Clean, modern UI with accessible design

## Installation

```bash
npm install
```

## Usage

To start the app in development mode:

```bash
npm start
```

or

```bash
npm run dev
```

## Project Structure

```
GitHub-Blogger/
├── main.js          # Electron main process
├── preload.js       # Electron preload script
├── index.html       # Main HTML page
├── styles.css       # Responsive styles
├── package.json     # Project dependencies
└── README.md        # This file
```

## Responsive Design

The app is fully responsive and supports:
- Desktop (1280px+)
- iPad Pro (1024px - 1366px)
- iPad (768px - 1024px)
- iPhone (up to 768px)
- iPhone SE and smaller devices (375px and below)

## Technologies

- Electron
- HTML5
- CSS3 with modern responsive design
- GitHub-inspired color scheme and typography
